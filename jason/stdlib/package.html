Provides all default internal actions of Jason.

BDI

  • {@link jason.stdlib.desire}: check agent's desires.
  • {@link jason.stdlib.drop_desire}: remove one agent's desire.
  • {@link jason.stdlib.drop_all_desires}: remove agent's desires.
  • {@link jason.stdlib.intend}: check agent's intentions.
  • {@link jason.stdlib.drop_intention}: remove one agent's intention.
  • {@link jason.stdlib.drop_all_intentions}: remove agent's intentions.
  • {@link jason.stdlib.current_intention}: get a description of the current intention.
  • {@link jason.stdlib.drop_event}: remove one event.
  • {@link jason.stdlib.drop_all_events}: remove events in the interpreter.
  • {@link jason.stdlib.succeed_goal}: abort some goal with success.
  • {@link jason.stdlib.fail_goal}: abort some goal with failure.
  • {@link jason.stdlib.suspend}: suspend intentions.
  • {@link jason.stdlib.resume}: resume suspended intentions.

Belief base

  • {@link jason.stdlib.abolish}: remove some beliefs.
  • {@link jason.stdlib.findall}: find all beliefs of some kind.
  • {@link jason.stdlib.count}: count the number of beliefs of some kind.

Plan Library

  • {@link jason.stdlib.add_plan}: add new plans.
  • {@link jason.stdlib.remove_plan}: remove a plan.
  • {@link jason.stdlib.plan_label}: get the label of a plan.
  • {@link jason.stdlib.relevant_plans}: get a list of plans.

Communication

  • {@link jason.stdlib.send}: send messages.
  • {@link jason.stdlib.broadcast}: broadcast messages.
  • {@link jason.stdlib.my_name}: get the agent's name.

Lists and Sets

  • {@link jason.stdlib.member}: list members.
  • {@link jason.stdlib.length}: size of lists.
  • {@link jason.stdlib.concat}: concat lists.
  • {@link jason.stdlib.delete}: delete members of a lists.
  • {@link jason.stdlib.reverse}: reverse lists.
  • {@link jason.stdlib.nth}: nth element of a lists.
  • {@link jason.stdlib.max}: maximum value of a lists.
  • {@link jason.stdlib.min}: minimum value of a lists.
  • {@link jason.stdlib.sort}: sort lists.
  • {@link jason.stdlib.list}: check whether an argument is a list.
  • {@link jason.stdlib.difference}: difference of sets.
  • {@link jason.stdlib.intersection}: intersection of sets.
  • {@link jason.stdlib.union}: union of sets.

String

  • {@link jason.stdlib.length}: size of strings.
  • {@link jason.stdlib.concat}: append strings.
  • {@link jason.stdlib.delete}: delete characters of a string.
  • {@link jason.stdlib.reverse}: reverse strings.
  • {@link jason.stdlib.substring}: test substrings of strings.
  • {@link jason.stdlib.string}: check whether an argument is a string.

Execution control

  • {@link jason.stdlib.conditional}: implementation of if.
  • {@link jason.stdlib.loop}: implementation of while.
  • {@link jason.stdlib.foreach}: implementation of for.

Meta programming

  • {@link jason.stdlib.atom}: check whether an argument is an atom.
  • {@link jason.stdlib.ground}: check whether an argument is ground.
  • {@link jason.stdlib.literal}: check whether an argument is a literal.
  • {@link jason.stdlib.list}: check whether an argument is a list.
  • {@link jason.stdlib.number}: check whether an argument is a number.
  • {@link jason.stdlib.string}: check whether an argument is a string.
  • {@link jason.stdlib.ground}: check whether an argument is ground.
  • {@link jason.stdlib.add_annot}: add an annotation in a literal.
  • {@link jason.stdlib.add_nested_source}: add a source in a literal.

Miscellaneous

  • {@link jason.stdlib.at}: add a future event.
  • {@link jason.stdlib.wait}: wait some event.
  • {@link jason.stdlib.create_agent}: create a new agent.
  • {@link jason.stdlib.kill_agent}: kill an agent.
  • {@link jason.stdlib.stopMAS}: stop all agents.
  • {@link jason.stdlib.date}: get the current date.
  • {@link jason.stdlib.time}: get the current time.
  • {@link jason.stdlib.fail}: an action that always fails.
  • {@link jason.stdlib.perceive}: force perception.
  • {@link jason.stdlib.range}: backtrack values in a range (used in for).
